# Git commit checklist (before pushing)

- [ ] Raw data is NOT tracked (confirm `git status` is clean; check `.gitignore`)
- [ ] Notebook outputs are stripped (optional but recommended)
- [ ] `INPUT_H5AD` uses a relative path under `data/raw/`
- [ ] `README.md` updated with your dataset description and contact
- [ ] `LICENSE` name updated
- [ ] `CITATION.cff` updated (authors + repo URL)
